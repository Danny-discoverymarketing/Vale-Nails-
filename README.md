# Vale Nails Website

Marketing website for Vale Nails, a nail salon in Guadalajara, Jalisco.

## Files

- `index.html` is the website page.
- `styles.css` contains the visual design.
- `script.js` handles the gallery lightbox and appointment calendar.
- `assets/gallery/` contains the salon images used across the site.

## Deploy on Vercel

1. Create a new GitHub repository.
2. Upload all files in this folder to the repository root.
3. In Vercel, choose **Add New Project** and import that GitHub repository.
4. Keep the default static-site settings and deploy.

No build command is required.
